## Full Description

```
Armada – Bid Your Price. Ride Armada. 🇯🇲

Armada is a Jamaican ride-share app where you set the price. Riders post a ride and bid; drivers counter. Agree on a fare and go.

Features:
• Bid & counter – negotiate fares that work for you
• Cash or card – pay with cash or PayPal
• Real-time tracking – see your driver on the map
• Emergency SOS – share location with contacts
• Food stops – add a pickup on your route
• Irie Coins – earn and redeem rewards

For riders: Request a ride, get bids, accept and pay.
For drivers: Go online, receive bids, accept and earn.
```

**Suggested screens:**
1. Login / phone auth
2. Rider: map with pickup/dropoff
3. Rider: bidding screen with driver offers
4. Driver: online / bid list
5. Active ride with map
6. Payment / receipt

**How to capture:**
```bash
# Run the app
npx expo start
# Then: npx expo run:android  or  npx expo run:ios
# Use device screenshot (Power+Vol Down on Android, Side+Vol Up on iOS)
```

Save screenshots to `assets/store-screenshots/` (create folder).
